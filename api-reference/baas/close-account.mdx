---
title: "Close Account"
description: "Close a BaaS bank account"
api: "DELETE https://api-sandbox.y.uno/v1/baas/accounts/{account_id}"
---

## Path parameters

<ParamField path="account_id" type="string" required>
  The BaaS account identifier to close.
</ParamField>
