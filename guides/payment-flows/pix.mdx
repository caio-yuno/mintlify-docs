## Overview

PIX is Brazil's instant payment system operated by the Central Bank. Payments are confirmed in seconds, 24/7.

## Prerequisites

- Country: `BR`
- Currency: `BRL`
- Customer document: CPF (11 digits) or CNPJ (14 digits) **required**
- PIX enabled in Dashboard > Payment Methods > Brazil

## Payment Flow

<Steps>
  <Step title="Create checkout session">
    ```json
    {
      "amount": {"currency": "BRL", "value": 100.00},
      "country": "BR",
      "merchant_order_id": "pix-001"
    }
    ```
  </Step>

  <Step title="Create PIX payment">
    ```json
    {
      "checkout_session": "session-id",
      "payment_method": {"type": "PIX"},
      "amount": {"currency": "BRL", "value": 100.00},
      "country": "BR",
      "customer": {
        "document": {"document_type": "CPF", "document_number": "12345678901"},
        "email": "customer@example.com"
      }
    }
    ```
  </Step>

  <Step title="Display QR code">
    The response includes a PIX QR code or copy-paste code. Display it to the customer.
  </Step>

  <Step title="Receive webhook notification">
    PIX payments are asynchronous. Set up [webhooks](/guides/webhooks/overview) to receive the payment confirmation (typically within seconds).
  </Step>
</Steps>

## Common Errors

| Error | Cause | Solution |
|-------|-------|----------|
| `customer.document is required for PIX` | Missing CPF/CNPJ | Add `customer.document` with valid CPF (11 digits) or CNPJ (14 digits) |
| Payment method not available | PIX not enabled | Enable PIX in Dashboard > Payment Methods > Brazil |
