## Key Features

- **Multi-provider orchestration** — Route payments through multiple providers with a single integration
- **Latin America coverage** — Support for PIX, Boleto, OXXO, PSE, SPEI, and more
- **SDK options** — Full Checkout, Lite Checkout, and Headless integration modes
- **Installments** — Native installment plan support for card payments
- **Webhooks** — Real-time payment status notifications
