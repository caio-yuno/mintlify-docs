## Airline Data

Include airline-specific information for travel-related transactions. This data may be required by certain acquirers and helps with fraud prevention for airline purchases.

## Schema

```json
{
  "airline": {
    "ticket_number": "0012345678901",
    "pnr": "ABC123",
    "departure_date": "2026-03-15",
    "origin_city": "GRU",
    "destination_city": "MIA",
    "carrier_code": "LA",
    "passenger_name": "DOE/JOHN",
    "legs": [
      {
        "departure_airport": "GRU",
        "arrival_airport": "MIA",
        "carrier_code": "LA",
        "flight_number": "8070",
        "departure_date": "2026-03-15",
        "class": "Y"
      }
    ]
  }
}
```

## Fields

| Field | Type | Description |
|-------|------|-------------|
| `ticket_number` | string | Airline ticket number (13 digits) |
| `pnr` | string | Passenger Name Record (booking reference) |
| `departure_date` | string | Departure date (YYYY-MM-DD) |
| `origin_city` | string | IATA code of origin city |
| `destination_city` | string | IATA code of destination city |
| `carrier_code` | string | IATA airline carrier code |
| `passenger_name` | string | Passenger name (LAST/FIRST format) |

## Leg Fields

| Field | Type | Description |
|-------|------|-------------|
| `departure_airport` | string | IATA departure airport code |
| `arrival_airport` | string | IATA arrival airport code |
| `carrier_code` | string | IATA carrier code for this leg |
| `flight_number` | string | Flight number |
| `departure_date` | string | Leg departure date (YYYY-MM-DD) |
| `class` | string | Booking class code |
