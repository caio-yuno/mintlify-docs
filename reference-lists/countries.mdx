## Supported Countries

The following countries are supported by Yuno for payment processing. Use the ISO 3166-1 alpha-2 code in API requests.

| Country | Code | Currency | Common Payment Methods |
|---------|------|----------|----------------------|
| Argentina | AR | ARS | Card, Bank Transfer |
| Bolivia | BO | BOB | Card |
| Brazil | BR | BRL | PIX, Boleto, Card |
| Chile | CL | CLP | Card, Bank Transfer |
| Colombia | CO | COP | PSE, Card, Cash (Efecty) |
| Costa Rica | CR | CRC | Card |
| Dominican Republic | DO | DOP | Card |
| Ecuador | EC | USD | Card |
| El Salvador | SV | USD | Card |
| Guatemala | GT | GTQ | Card |
| Honduras | HN | HNL | Card |
| Mexico | MX | MXN | OXXO, SPEI, Card |
| Nicaragua | NI | NIO | Card |
| Panama | PA | USD | Card |
| Paraguay | PY | PYG | Card |
| Peru | PE | PEN | Card, Bank Transfer |
| Puerto Rico | PR | USD | Card |
| United States | US | USD | Card |
| Uruguay | UY | UYU | Card |
| Venezuela | VE | VES | Card |

<Note>
Payment method availability varies by country and provider configuration. Verify enablement in the Yuno Dashboard for your specific setup.
</Note>
