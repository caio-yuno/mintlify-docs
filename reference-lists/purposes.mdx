## Payout Purposes

Use these purpose codes in the `purpose` field when creating payouts. Purpose codes help categorize payouts for regulatory compliance.

| Purpose | Description |
|---------|-------------|
| `SALARY` | Employee salary payment |
| `COMMISSION` | Sales commission |
| `REFUND` | Customer refund |
| `BONUS` | Performance or signing bonus |
| `LOAN` | Loan disbursement |
| `SUPPLIER` | Supplier payment |
| `VENDOR` | Vendor payment |
| `RENT` | Rental payment |
| `ROYALTY` | Royalty payment |
| `DIVIDEND` | Dividend distribution |
| `TRANSFER` | Internal transfer |
| `OTHER` | Other purpose |

<Note>
Required purpose codes may vary by country and provider. Check your provider's documentation for country-specific requirements.
</Note>
