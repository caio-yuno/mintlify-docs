## Shipping Object

Include shipping information in payment requests when applicable. Shipping data helps with fraud detection and may be required by certain payment providers.

## Schema

```json
{
  "shipping": {
    "address": {
      "address_line_1": "123 Main Street",
      "address_line_2": "Apt 4B",
      "city": "Sao Paulo",
      "state": "SP",
      "zip_code": "01310-100",
      "country": "BR"
    },
    "first_name": "John",
    "last_name": "Doe",
    "phone": "+5511999999999",
    "method": "STANDARD"
  }
}
```

## Address Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `address_line_1` | string | Yes | Street address |
| `address_line_2` | string | No | Apartment, suite, unit |
| `city` | string | Yes | City name |
| `state` | string | Yes | State or province code |
| `zip_code` | string | Yes | Postal or ZIP code |
| `country` | string | Yes | ISO 3166-1 alpha-2 country code |

## Shipping Methods

| Method | Description |
|--------|-------------|
| `STANDARD` | Standard shipping (3-7 business days) |
| `EXPRESS` | Express shipping (1-2 business days) |
| `SAME_DAY` | Same-day delivery |
| `PICKUP` | In-store or local pickup |
| `DIGITAL` | Digital delivery (no physical shipping) |
